This is a project for USP - Universidade de S~ao Paulo

It is a new Interface for the research Database BRnpDB - Brazilian Natural Products Database, based on the Ontology NubbeKG: https://nubbekg.aksw.org/ontology/index.html

The script to transfer the data from the relational database brnpdb to RDF according to the NubbeKG ontology (using .ttl) is included as well

###Still under construction and to be published here: https://brnpdb.fcfrp.usp.br/
